# 🧠 Log Anomaly Detector – Carine Jackson

This project parses security logs (syslog, Semgrep, Trivy) and flags potential anomalies using rule-based detection. Built for visibility and early-stage security analytics.

## 🚀 Features
- Parses syslogs and JSON security reports
- Basic rule-based anomaly detection (YAML)
- Visualizes issues by time, severity
- Designed for portfolio and practical detection practice
